# References

This section lists credible resources used and recommended for learning and working with **Flutter**. These references support the Flutter Toolkit and can be used for academic work, projects, and further practice.

---

## Official Documentation

1. Google. (n.d.). *Flutter documentation*. [https://docs.flutter.dev](https://docs.flutter.dev)

   > The official and most reliable source for Flutter setup, widgets, layouts, and best practices.

2. Google. (n.d.). *Dart programming language*. [https://dart.dev](https://dart.dev)

   > Official documentation for Dart, the language used to build Flutter applications.

---

## Frameworks & Tools

3. Google. (n.d.). *Flutter SDK*. [https://flutter.dev](https://flutter.dev)

   > Official Flutter SDK download and overview of supported platforms.

4. Android Developers. (n.d.). *Android Studio documentation*. [https://developer.android.com/studio](https://developer.android.com/studio)

   > Guidance on using Android Studio as an IDE for Flutter development.

5. Microsoft. (n.d.). *Visual Studio Code documentation*. [https://code.visualstudio.com/docs](https://code.visualstudio.com/docs)

   > Documentation for using VS Code with Flutter and Dart extensions.

## Learning Resources

6. Flutter.dev. (n.d.). *Flutter codelabs*. [https://docs.flutter.dev/codelabs](https://docs.flutter.dev/codelabs)

7. Flutter.dev. (n.d.). *Cookbook: Useful Flutter samples*. [https://docs.flutter.dev/cookbook](https://docs.flutter.dev/cookbook)

   > Practical examples for common Flutter tasks such as navigation, forms, and state handling.

8. Udemy. (n.d.). *Flutter development courses*. [https://www.udemy.com](https://www.udemy.com)

   > Online courses for structured Flutter learning (use instructor-reviewed courses).
